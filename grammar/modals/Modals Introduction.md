> Этот файл охватывает все модальные глаголы английского языка. Модальные глаголы выражают отношение говорящего к действию — возможность, необходимость, разрешение, запрет, вероятность, обязанность и др.

Модальные глаголы не спрягаются — у них нет окончания -s для третьего лица, нет формы -ing, нет инфинитива с to. После них всегда стоит bare infinitive (глагол без to). У большинства нет всех временных форм — для прошедшего и будущего используются специальные конструкции.


[[Can и Could — умение, возможность, разрешение]]
[[May и Might — разрешение, вероятность]]
[[Must — необходимость, запрет, уверенное предположение]]
[[Shall и Should — предложения, советы, обязанность]]
[[Will и Would — будущее, вежливость, привычки]]
[[Ought To — моральная обязанность]]
[[Need и Dare — необходимость и смелость]]


**Общие свойства модальных глаголов:**

| Свойство | Пояснение |
|---|---|
| Нет -s в 3-м лице | He can (не: he cans), she must (не: she musts) |
| Нет -ing формы | Нельзя: *I am canning*, *she is musting* |
| Нет to-infinitive | Нельзя: *to can*, *to must* |
| После них — bare infinitive | You must go. / She can swim. / He should call. |
| Отрицание без do | I can't (не: I don't can), he mustn't (не: he doesn't must) |
| Вопрос без do | Can you help? (не: Do you can help?) |

**Сравнительная таблица значений:**

| Значение | Глаголы |
|---|---|
| Умение / способность | can, could, be able to |
| Возможность | can, could, may, might |
| Разрешение | can, could, may, might |
| Запрет | can't, mustn't |
| Необходимость / обязанность | must, have to, should, ought to, need to |
| Отсутствие необходимости | don't have to, needn't |
| Вероятность / предположение | must, can't, should, may, might, could |
| Совет | should, ought to, had better |
| Просьба | can, could, will, would |
