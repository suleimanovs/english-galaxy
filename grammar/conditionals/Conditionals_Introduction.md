> Этот файл охватывает все типы условных предложений английского языка, а также конструкции с wish. Условные предложения описывают ситуации «если... то...» — реальные, возможные, маловероятные или невозможные.

В русском условные предложения строятся гибко: «если бы я знал» одинаково работает и для маловероятного настоящего, и для невозможного прошлого. В английском каждый тип условия имеет свою чёткую структуру, и смешивать их нельзя — от выбора времени зависит смысл.

**Сравнительная таблица всех типов:**

| Тип | If-clause | Main clause | Смысл |
|---|---|---|---|
| **Zero** | if + Present Simple | Present Simple | Всегда истинно, закономерность |
| **First** | if + Present Simple | will + V | Реальное условие в будущем |
| **Second** | if + Past Simple | would + V | Нереальное / маловероятное настоящее или будущее |
| **Third** | if + Past Perfect | would have + V3 | Нереальное прошлое |
| **Mixed** | if + Past Perfect | would + V | Прошлое условие → настоящий результат |
| **Mixed** | if + Past Simple | would have + V3 | Настоящее условие → прошлый результат |
