First Conditional это условные предложения о **реальных и возможных ситуациях в будущем**. Условие реально — говорящий считает его выполнимым. Результат — то, что произойдёт, если условие выполнится.

Это самый «оптимистичный» тип условия: говорящий допускает, что это вполне может случиться.

---

**Структура:**

```
If + Present Simple  ,  will + V (base form)
```

В if-clause — Present Simple, даже если речь о будущем (это то самое правило: после if будущее не используется). В главной части — will + глагол.

- If it rains tomorrow, I'll stay home. — Если завтра пойдёт дождь, я останусь дома.
- If you study hard, you will pass the exam. — Если ты будешь усердно учиться, ты сдашь экзамен.
- I'll call you if I'm late. — Я позвоню тебе, если опоздаю.
- If she doesn't come, we'll start without her. — Если она не придёт, мы начнём без неё.

---

#### Когда использовать First Conditional

**1. Реальные условия и их последствия в будущем**

- If you miss the bus, you'll be late. — Если пропустишь автобус, опоздаешь.
- If we leave now, we'll catch the train. — Если уйдём сейчас, успеем на поезд.
- If the weather is good, we'll have a picnic. — Если погода будет хорошей, устроим пикник.

**2. Предупреждения**

- If you touch that wire, you'll get electrocuted. — Если дотронешься до этого провода, тебя ударит током.
- If you don't take your medicine, you won't get better. — Если не будешь принимать лекарство, не поправишься.

**3. Обещания и предложения**

- If you help me, I'll buy you dinner. — Если поможешь мне, я куплю тебе ужин.
- If you need anything, I'll be happy to help. — Если тебе что-нибудь понадобится, я с удовольствием помогу.

**4. Просьбы (с imperative в главной части)**

В главной части вместо will может стоять повелительное наклонение.

- If you see her, tell her to call me. — Если увидишь её, скажи ей, чтобы позвонила мне.
- If you finish early, come and help me. — Если закончишь пораньше, приходи помоги мне.

---

**Варианты в главной части**

Вместо will можно использовать другие модальные глаголы в зависимости от оттенка:

| Главная часть | Оттенок | Пример |
|---|---|---|
| will + V | уверенный результат | If she studies, she will pass. |
| can + V | возможность | If you practise, you can improve. |
| may / might + V | менее уверенный результат | If it rains, we might cancel it. |
| should + V | рекомендация | If you feel ill, you should see a doctor. |
| imperative | инструкция, просьба | If you see him, tell him to call me. |

---

**Сигнальные слова**

Вместо if в First Conditional можно использовать другие союзы:

| Союз | Перевод | Пример |
|---|---|---|
| if | если | If it rains, I'll stay home. |
| unless | если не / до тех пор пока не | Unless you hurry, you'll be late. |
| as long as | при условии что | I'll help as long as you try. |
| provided (that) | при условии что | I'll come provided that you invite me. |
| in case | на случай если | Take an umbrella in case it rains. |
| when | когда (уверенный результат) | When I finish, I'll call you. |

[!] **unless** = **if...not**: *Unless you hurry* = *If you don't hurry*.

[!] **in case** ≠ **if**: *Take an umbrella in case it rains* — возьми зонт на всякий случай (заранее, независимо от того, пойдёт ли дождь). *Take an umbrella if it rains* — возьми зонт, если пойдёт дождь (условие).

---

**Типичные ошибки русскоязычных**

Ошибка 1: Will в if-clause

Самая распространённая ошибка — в русском «если пойдёт дождь» переводится с будущим, в английском после if нужен Present Simple.

| Неправильно | Правильно |
|---|---|
| If it will rain, I'll stay home. | If it rains, I'll stay home. |
| If she will come, tell me. | If she comes, tell me. |
| If you will help me, I'll be grateful. | If you help me, I'll be grateful. |

Ошибка 2: Путать First и Second Conditional

First — реальная ситуация (может случиться). Second — нереальная или маловероятная (гипотеза).

- *If I win the lottery, I'll buy a house.* — First: реально, я мог бы выиграть.
- *If I won the lottery, I would buy a house.* — Second: маловероятно, просто фантазия.

Ошибка 3: Путать unless и if not

Unless означает «если только не» — условие отрицательное само по себе, добавлять not не нужно.

| Неправильно | Правильно |
|---|---|
| Unless you don't hurry, you'll be late. | Unless you hurry, you'll be late. |

---

**Сравнение: First vs Second Conditional**

| | First Conditional | Second Conditional |
|---|---|---|
| **Смысл** | Реальное, возможное условие | Нереальное, гипотетическое условие |
| **If-clause** | Present Simple | Past Simple |
| **Main clause** | will + V | would + V |
| **Пример** | If it rains, I'll stay home. | If it rained, I would stay home. |
| **Подтекст** | Это вполне может случиться | Это маловероятно или невозможно |
