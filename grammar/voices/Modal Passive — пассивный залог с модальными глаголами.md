Пассивный залог с модальными глаголами (Modal Passive) сочетает модальные глаголы с пассивной конструкцией. Это позволяет говорить о возможности, необходимости, обязанности — при этом фокусируясь на объекте действия, а не на деятеле.

---

**Структура:**

```
Подлежащее  +  модальный глагол  +  be  +  V3 (Past Participle)
```

- The report **must be finished** by Friday. — Отчёт должен быть закончен к пятнице.
- This **can be done** easily. — Это можно сделать легко.
- The rules **should be followed**. — Правила нужно соблюдать.

---

#### Все модальные глаголы в пассиве

| Модальный глагол | Активный залог | Пассивный залог |
|---|---|---|
| **can** | They can fix it. | It **can be fixed**. |
| **could** | They could see it. | It **could be seen**. |
| **may** | They may cancel the event. | The event **may be cancelled**. |
| **might** | They might change the plan. | The plan **might be changed**. |
| **must** | You must finish the report. | The report **must be finished**. |
| **should** | You should clean the room. | The room **should be cleaned**. |
| **will** | They will build a new bridge. | A new bridge **will be built**. |
| **would** | They would accept the offer. | The offer **would be accepted**. |
| **have to** | You have to pay the fine. | The fine **has to be paid**. |
| **ought to** | You ought to respect the rules. | The rules **ought to be respected**. |
| **need to** | You need to sign the form. | The form **needs to be signed**. |

---

#### Modal + have been + V3 — прошлое в пассиве

Для прошлых событий с модальными:

```
Подлежащее  +  модальный глагол  +  have been  +  V3
```

| Конструкция | Значение | Пример |
|---|---|---|
| **must have been + V3** | Уверенное предположение о прошлом | The letter **must have been lost**. — Письмо, должно быть, потеряли. |
| **could have been + V3** | Возможность в прошлом | The mistake **could have been avoided**. — Ошибки можно было избежать. |
| **should have been + V3** | Следовало бы (но не сделали) | The work **should have been completed** last week. — Работу следовало завершить на прошлой неделе. |
| **might have been + V3** | Вероятность в прошлом | He **might have been injured** in the accident. — Он, возможно, был ранен в аварии. |
| **would have been + V3** | Условный пассив (в Third Conditional) | The project **would have been approved** if... — Проект был бы одобрен, если бы... |
| **needn't have been + V3** | Не нужно было (но сделали) | The door **needn't have been locked**. — Дверь не нужно было запирать. |

#### Примеры по контексту

**Правила и инструкции:**
- Seat belts **must be worn** at all times. — Ремни безопасности должны быть пристёгнуты.
- Mobile phones **must not be used** during the exam. — Мобильные телефоны не должны использоваться на экзамене.
- This form **should be filled** in before the interview. — Эту форму нужно заполнить перед интервью.

**Возможность и способность:**
- This problem **can be solved** in many ways. — Эту проблему можно решить разными способами.
- The building **can be seen** from the hill. — Здание видно с холма.
- The data **cannot be recovered**. — Данные невозможно восстановить.

**Предположения:**
- He **may be promoted** this year. — Его, возможно, повысят в этом году.
- The flight **might be delayed**. — Рейс может быть задержан.
- The package **should be delivered** by tomorrow. — Посылка должна быть доставлена к завтрашнему дню.

**Будущее:**
- A new hospital **will be built** next year. — В следующем году будет построена новая больница.
- The results **will be announced** on Monday. — Результаты будут объявлены в понедельник.

---

**Типичные ошибки русскоязычных**

Ошибка 1: Пропуск be в конструкции

| Неправильно | Правильно |
|---|---|
| It must finished by Friday. | It must **be** finished by Friday. |
| The room should cleaned. | The room should **be** cleaned. |
| This can done easily. | This can **be** done easily. |

Ошибка 2: Неправильная форма причастия

| Неправильно | Правильно |
|---|---|
| It must be finish. | It must be finish**ed**. |
| The door can be open. | The door can be open**ed**. |

Ошибка 3: Активный залог там, где нужен пассивный

Когда деятель неизвестен или неважен:

| Менее естественно | Более естественно |
|---|---|
| Someone must repair this road. | This road **must be repaired**. |
| They should inform the police. | The police **should be informed**. |
| You can't use this room now. | This room **can't be used** now. |
