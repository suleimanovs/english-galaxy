> В английском 44 звука: 20 гласных и 24 согласных. В русском 42 звука, но распределение совсем другое: всего 6 гласных. Многие английские звуки не имеют аналогов в русском.

Ниже все звуки с IPA (международный фонетический алфавит), приближённым описанием для русскоязычных и примерами.

---

## Гласные (Vowels) -- 20 звуков

### Короткие гласные (Short vowels) -- 7 звуков

| IPA | Приближение | Пример | Транскрипция |
|---|---|---|---|
| /ɪ/ | Короткое "и", ближе к "ы" | b**i**t, s**i**t, f**i**sh | /bɪt/, /sɪt/, /fɪʃ/ |
| /e/ | Как русское "э" | b**e**d, r**e**d, s**e**t | /bed/, /red/, /set/ |
| /ae/ | Между "э" и "а", рот широко открыт | c**a**t, b**a**d, h**a**nd | /kaet/, /baed/, /haend/ |
| /ʌ/ | Короткое "а", ненапряжённое | c**u**t, b**u**s, l**u**ck | /kʌt/, /bʌs/, /lʌk/ |
| /ɒ/ | Короткое округлённое "о" (British) | h**o**t, d**o**g, l**o**t | /hɒt/, /dɒɡ/, /lɒt/ |
| /ʊ/ | Короткое "у", губы чуть округлены | p**u**t, b**oo**k, g**oo**d | /pʊt/, /bʊk/, /ɡʊd/ |
| /ə/ | Schwa -- самый частый звук. Безударное "э", едва слышное | **a**bout, sof**a**, teach**e**r | /əˈbaʊt/, /ˈsəʊfə/, /ˈtiːtʃə/ |

Schwa /ə/ -- ключевой звук. Он появляется в КАЖДОМ безударном слоге. Русскоязычные произносят безударные слоги слишком чётко. Англичане их "проглатывают" до schwa.

---

### Длинные гласные (Long vowels) -- 5 звуков

| IPA | Приближение | Пример | Транскрипция |
|---|---|---|---|
| /iː/ | Длинное "и" | s**ee**, f**ee**l, t**ea** | /siː/, /fiːl/, /tiː/ |
| /ɑː/ | Длинное открытое "а" | c**ar**, f**ar**, h**ear**t | /kɑː/, /fɑː/, /hɑːt/ |
| /ɔː/ | Длинное "о" | d**oor**, f**our**, w**al**k | /dɔː/, /fɔː/, /wɔːk/ |
| /uː/ | Длинное "у" | f**oo**d, bl**ue**, sh**oe** | /fuːd/, /bluː/, /ʃuː/ |
| /ɜː/ | Среднее между "о" и "э", губы нейтральны | b**ir**d, w**or**d, h**er** | /bɜːd/, /wɜːd/, /hɜː/ |

/ɜː/ -- не имеет аналога в русском. Губы не округлены, язык в центре рта. Как если бы вы начали говорить "о", но не округлили губы.

---

### Дифтонги (Diphthongs) -- 8 звуков

Дифтонг = два гласных звука, сливающихся в один слог. Начинается с одного и скользит к другому.

| IPA | Приближение | Пример | Транскрипция |
|---|---|---|---|
| /eɪ/ | "эй" | d**ay**, pl**ay**, m**a**ke | /deɪ/, /pleɪ/, /meɪk/ |
| /aɪ/ | "ай" | t**i**me, n**i**ght, b**uy** | /taɪm/, /naɪt/, /baɪ/ |
| /ɔɪ/ | "ой" | b**oy**, t**oy**, j**oi**n | /bɔɪ/, /tɔɪ/, /dʒɔɪn/ |
| /aʊ/ | "ау" | h**ou**se, n**ow**, **ou**t | /haʊs/, /naʊ/, /aʊt/ |
| /əʊ/ | "оу" (British) | g**o**, h**o**me, ph**o**ne | /ɡəʊ/, /həʊm/, /fəʊn/ |
| /ɪə/ | "иэ" | h**ere**, n**ear**, id**ea** | /hɪə/, /nɪə/, /aɪˈdɪə/ |
| /eə/ | "эа" | wh**ere**, c**are**, h**air** | /weə/, /keə/, /heə/ |
| /ʊə/ | "уэ" | t**our**, p**ure**, s**ure** | /tʊə/, /pjʊə/, /ʃʊə/ |

Ошибка русскоязычных: произносить дифтонг как два отдельных звука. "Make" должен быть /meɪk/ (один слог), а не "мэ-йк" (два удара).

## Согласные (Consonants) -- 24 звука

### Проблемные согласные для русскоязычных

| IPA | Как произнести | Пример | Ошибка русскоязычных |
|---|---|---|---|
| /θ/ | Кончик языка между зубами, дуть. Глухой | **th**ink, **th**ree, ba**th** | Заменяют на "с" или "ф": *sink* вместо *think* |
| /ð/ | Кончик языка между зубами, дуть. Звонкий | **th**is, **th**at, mo**th**er | Заменяют на "з" или "д": *zis* вместо *this* |
| /w/ | Губы в трубочку, как начало "у", затем гласный | **w**ater, **w**hy, **w**ork | Заменяют на "в": *vater* вместо *water* |
| /v/ | Верхние зубы касаются нижней губы | **v**ery, lo**v**e, li**v**e | Путают с /w/ |
| /r/ | Язык загнут назад, НЕ вибрирует | **r**ed, **r**ight, ca**r** | Произносят русское раскатистое "р" |
| /ŋ/ | Носовой звук, как "н" но в горле | si**ng**, thi**nk**, lo**ng** | Произносят "нг" как два звука: *sin-g* |
| /h/ | Лёгкий выдох из горла | **h**ouse, **h**ello, **h**appy | Произносят как русское "х" (слишком грубо) |
| /dʒ/ | Мягкое "дж" | **j**ob, a**g**e, bri**dg**e | Произносят слишком твёрдо |
| /tʃ/ | Как русское "ч" | **ch**air, tea**ch**, wat**ch** | Обычно нет проблем |

## Минимальные пары (Minimal Pairs)

Пары слов, отличающихся одним звуком. Тренировка на различение.

**/ ɪ / vs / iː / (короткое vs длинное "и"):**
ship / sheep, sit / seat, bit / beat, fill / feel, hit / heat, live / leave, sin / seen, fit / feet

**/ e / vs / ae / ("э" vs "а-э"):**
bed / bad, set / sat, men / man, pen / pan, beg / bag, met / mat, head / had

**/ ʌ / vs / ɑː / (короткое vs длинное "а"):**
cut / cart, hut / heart, bud / bard, duck / dark, luck / lark

**/ ɒ / vs / ɔː / (короткое vs длинное "о"):**
cot / caught, shot / short, pot / port, not / nought, fox / forks

**/ ʊ / vs / uː / (короткое vs длинное "у"):**
pull / pool, full / fool, look / Luke, could / cooed

**/ θ / vs / s / (th vs s):**
think / sink, thick / sick, path / pass, math / mass, thumb / sum, three / see (tree)

**/ ð / vs / z / (th vs z):**
then / zen, they / zay, breathe / breeze, clothe / close

**/ w / vs / v /:**
wine / vine, wet / vet, west / vest, wail / veil, worse / verse

**/ b / vs / v /:**
berry / very, best / vest, boat / vote, ban / van

**/ l / vs / r /:**
light / right, long / wrong, lead / read, fly / fry, glass / grass, collect / correct
