Множественное число (Plural Forms) в английском образуется по правилам, но имеет множество исключений. Неправильные формы нужно запоминать, а некоторые слова вообще не меняются.

---

#### Правильное образование множественного числа

**1. Основное правило: + s**

- book → books, cat → cats, day → days, phone → phones

**2. После -s, -ss, -sh, -ch, -x, -z: + es**

- bus → buses, glass → glasses, brush → brushes, watch → watches, box → boxes, quiz → quizzes

**3. После -o:**

- С -es (живые, старые слова): potato → potatoes, tomato → tomatoes, hero → heroes, echo → echoes
- С -s (заимствования, музыка, сокращения): photo → photos, piano → pianos, video → videos, zoo → zoos, memo → memos

**4. После согласная + y: y → ies**

- city → cities, baby → babies, story → stories, party → parties
- [!] После гласная + y: просто + s: boy → boys, day → days, key → keys, toy → toys

**5. После -f / -fe: f → ves**

- knife → knives, wife → wives, life → lives, leaf → leaves, shelf → shelves, half → halves, wolf → wolves, thief → thieves
- [!] Исключения (просто + s): roof → roofs, chief → chiefs, belief → beliefs, cliff → cliffs, proof → proofs

---

#### Неправильные формы (Irregular Plurals)

Эти формы нужно запоминать — правила не применяются:

| Единственное | Множественное | Перевод |
|---|---|---|
| man | men | мужчина / мужчины |
| woman | women | женщина / женщины |
| child | children | ребёнок / дети |
| foot | feet | стопа / стопы |
| tooth | teeth | зуб / зубы |
| goose | geese | гусь / гуси |
| mouse | mice | мышь / мыши |
| louse | lice | вошь / вши |
| ox | oxen | бык / быки |
| person | people | человек / люди |

[!] **Person → people** — это основная форма. *Persons* используется только в юридическом языке.

---

#### Неизменяемые формы (одинаковые в ед. и мн. числе)

| Слово | Пример |
|---|---|
| sheep | one sheep → five sheep |
| fish | one fish → many fish |
| deer | one deer → several deer |
| aircraft | one aircraft → two aircraft |
| species | one species → many species |
| series | one series → two series |
| means | one means → several means |
| crossroads | one crossroads → two crossroads |

[!] **Fish**: *fishes* используется, когда речь идёт о разных видах рыб: *The seas contain many different fishes.*

---

#### Заимствования из латинского и греческого

| Единственное | Множественное | Пример |
|---|---|---|
| analysis | analyses | data analysis → data analyses |
| basis | bases | on this basis → on these bases |
| crisis | crises | an economic crisis → economic crises |
| thesis | theses | my thesis → their theses |
| phenomenon | phenomena | a strange phenomenon → strange phenomena |
| criterion | criteria | the main criterion → the main criteria |
| datum | data | — (data используется и как ед., и как мн.) |
| medium | media | social media (формально мн. число) |
| formula | formulae / formulas | оба варианта допустимы |
| focus | foci / focuses | оба варианта допустимы |
| index | indices / indexes | indices (математика), indexes (книга) |

---

#### Существительные только во множественном числе (Pluralia tantum)

Эти слова **всегда множественного числа** и используются с **are/were/have**:

| Слово | Перевод | Пример |
|---|---|---|
| trousers | брюки | These trousers **are** too long. |
| jeans | джинсы | My jeans **are** new. |
| shorts | шорты | Where **are** my shorts? |
| glasses | очки | Her glasses **are** on the table. |
| scissors | ножницы | The scissors **are** sharp. |
| stairs | лестница | The stairs **are** steep. |
| clothes | одежда | These clothes **are** expensive. |
| goods | товары | The goods **have** arrived. |
| police | полиция | The police **are** looking for him. |

Чтобы сказать «один»: **a pair of** trousers / scissors / glasses / jeans.

---

#### Существительные только в единственном числе

Всегда единственного числа и используются с **is/was/has**:

- news: The news **is** good. (не *are*)
- mathematics / physics / economics: Mathematics **is** my favourite subject.
- measles / mumps: Measles **is** dangerous.
- the United States / the Netherlands: The United States **is** a large country. (как одна страна)

**Типичные ошибки русскоязычных**

Ошибка 1: People как единственное число

| Неправильно | Правильно |
|---|---|
| People is friendly here. | People are friendly here. |
| A people came to visit. | A person came to visit. |

Ошибка 2: Двойное множественное

| Неправильно | Правильно |
|---|---|
| childrens | children |
| womens | women |
| mices | mice |

Ошибка 3: Неправильное множественное у заимствований

| Неправильно | Правильно |
|---|---|
| one criteria | one criterion |
| one phenomena | one phenomenon |
| one media | one medium |
| several datum | several data |

Ошибка 4: News / Police — число глагола

| Неправильно | Правильно |
|---|---|
| The news are bad. | The news is bad. |
| The police is here. | The police are here. |
