> Словарь 200 самых важных фразовых глаголов английского языка. Крупные базовые глаголы (5+ вариаций) вынесены в отдельные файлы. Глаголы с 1–4 вариациями собраны в общей таблице.

Фразовые глаголы — одна из главных сложностей английского для русскоязычных. Один базовый глагол с разными частицами даёт совершенно разные значения. Этот словарь охватывает ~200 самых частотных фразовых глаголов, сгруппированных по базовому глаголу.
