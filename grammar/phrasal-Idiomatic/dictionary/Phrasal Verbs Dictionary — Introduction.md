> Словарь самых важных фразовых глаголов английского языка. Крупные базовые глаголы (5+ вариаций) вынесены в отдельные файлы. Глаголы с 1–4 вариациями собраны в общей таблице. Каждое значение — отдельная строка.

Фразовые глаголы — одна из главных сложностей английского для русскоязычных. Один базовый глагол с разными частицами даёт совершенно разные значения. Этот словарь охватывает самые частотные фразовые глаголы, сгруппированные по базовому глаголу.
