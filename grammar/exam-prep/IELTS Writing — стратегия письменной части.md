> Writing -- часть IELTS, которая вызывает больше всего проблем у русскоязычных. Здесь пошаговые шаблоны для Task 1 и Task 2.

---

## Task 1 Academic -- описание графика

### Структура (4 абзаца, ~150-180 слов)

```
Paragraph 1: Introduction (перефразировать задание)
  The [chart/graph/table] shows/illustrates/compares...

Paragraph 2: Overview (главные тренды -- 2-3 предложения)
  Overall, it is clear that...
  The most notable trend is...

Paragraph 3: Detail 1 (конкретные данные с цифрами)

Paragraph 4: Detail 2 (конкретные данные с цифрами)
```

Правило: Overview обязателен. Без него потолок балла -- 5.0.

### Фразы для описания трендов

**Рост:**

| Глагол | Существительное | Пример |
|---|---|---|
| increase | an increase | Sales **increased** from 100 to 200. / There was **an increase** in sales. |
| rise | a rise | Prices **rose** sharply. / There was **a sharp rise** in prices. |
| grow | growth | The economy **grew** by 3%. / There was **growth** of 3%. |
| go up | -- | Temperatures **went up** in July. |
| climb | -- | Unemployment **climbed** to 8%. |
| surge | a surge | Demand **surged** in December. |

**Падение:**

| Глагол | Существительное |
|---|---|
| decrease | a decrease |
| decline | a decline |
| fall | a fall |
| drop | a drop |
| go down | -- |
| plummet | a plummet (резкое падение) |

**Стабильность:**

| Фразы |
|---|
| remain stable / steady / constant / unchanged |
| level off / flatten out |
| stay at the same level |
| fluctuate (колебаться) |

**Модификаторы (скорость/масштаб):**

| Наречие (с глаголом) | Прилагательное (с сущ.) |
|---|---|
| sharply, dramatically, significantly | a sharp, dramatic, significant increase |
| gradually, steadily | a gradual, steady rise |
| slightly, marginally | a slight, marginal decline |
| rapidly | a rapid growth |

**Фразы для цифр:**

| Фраза | Пример |
|---|---|
| from X to Y | Sales rose **from** 100 **to** 300. |
| by X | Revenue increased **by** 50%. |
| to X | The number fell **to** 200. |
| at X | The figure stood **at** 1,000. |
| approximately / about / around | **Approximately** 40% of respondents agreed. |
| just over / just under | **Just over** half the population... |

### Фразы для сравнения

| Фраза | Пример |
|---|---|
| whereas / while | Sales in the UK rose, **whereas** in Germany they fell. |
| compared to/with | **Compared to** 2010, the figure doubled. |
| in contrast | **In contrast**, the female rate was much lower. |
| similarly | **Similarly**, France showed the same trend. |
| X times as much/many as | China produced **three times as much** steel as India. |
| the highest / lowest | Japan had **the highest** rate at 85%. |

## Task 1 General Training -- письмо

### Типы писем

| Тип | Стиль | Обращение |
|---|---|---|
| Formal (жалоба, запрос в компанию) | Dear Sir or Madam / Dear Mr Smith | Yours faithfully / Yours sincerely |
| Semi-formal (коллеге, преподавателю) | Dear [Name] | Kind regards |
| Informal (другу) | Dear [Name] / Hi [Name] | Best wishes / Take care |

### Структура (3 абзаца)

```
Opening: объяснить зачем пишешь
  I am writing to... / I'm writing to let you know that...

Body: детали (1-2 абзаца)
  Подробности ситуации, просьбы, предложения

Closing: что ожидаешь дальше
  I look forward to hearing from you. / Hope to hear from you soon.
```

## Task 2 -- эссе

### 5 типов заданий с шаблонами

#### Тип 1: Opinion (Agree/Disagree)

Задание: "Some people think that... Do you agree or disagree?"

```
Introduction (2-3 предложения):
  [Перефразировать тему]. In my opinion, [ваш тезис].

Body 1 (главный аргумент ЗА вашу позицию):
  One of the main reasons why I believe this is...
  [Объяснение + пример]

Body 2 (второй аргумент):
  Furthermore, / Another important point is that...
  [Объяснение + пример]

Body 3 (контраргумент + ваш ответ -- опционально):
  Admittedly, some people argue that... However, I would counter that...

Conclusion:
  In conclusion, [перефразировать тезис]. [Финальная мысль].
```

#### Тип 2: Discussion + Opinion

Задание: "Discuss both views and give your own opinion."

```
Introduction:
  [Перефразировать тему]. While some people believe [view A],
  others argue [view B]. In my view, [ваша позиция].

Body 1 (View A):
  On the one hand, [аргументы первой стороны].

Body 2 (View B):
  On the other hand, [аргументы второй стороны].

Body 3 (ваше мнение):
  In my opinion, [ваша позиция + почему].

Conclusion:
  In conclusion, although both views have merit, I believe that...
```

#### Тип 3: Problem-Solution

Задание: "What problems does this cause? What solutions can you suggest?"

```
Introduction:
  [Описать ситуацию]. This essay will examine the problems
  caused by this and propose possible solutions.

Body 1 (проблемы):
  One of the main problems is... Another significant issue is...

Body 2 (решения):
  A possible solution to the first problem would be...
  In addition, [второе решение].

Conclusion:
  In conclusion, while [проблема] poses serious challenges,
  [решения] could help address them effectively.
```

#### Тип 4: Two-part question

Задание: "Why is this happening? Is this a positive or negative development?"

```
Introduction:
  [Перефразировать]. This essay will explore the reasons behind
  this trend and evaluate whether it is a positive or negative development.

Body 1 (ответ на первый вопрос):
  There are several reasons for this phenomenon. Firstly,...

Body 2 (ответ на второй вопрос):
  In my view, this is largely a [positive/negative] development because...

Conclusion:
  To sum up, [причины] and [оценка].
```

#### Тип 5: Advantages/Disadvantages

Задание: "What are the advantages and disadvantages?"

```
Introduction:
  [Перефразировать]. This essay will discuss the benefits
  and drawbacks of [тема].

Body 1 (advantages):
  One key advantage is... Furthermore,...

Body 2 (disadvantages):
  However, there are also drawbacks. One major disadvantage is...

Conclusion:
  On balance, although there are clear disadvantages,
  the advantages [outweigh/do not outweigh] them because...
```

## Фразы, которые поднимают балл

### Для Coherence & Cohesion (связность)

| Функция | Фразы |
|---|---|
| Добавление | Moreover, Furthermore, In addition, Not only... but also |
| Контраст | However, Nevertheless, On the other hand, Despite this |
| Пример | For instance, A case in point is, To illustrate |
| Следствие | As a result, Consequently, Therefore, This means that |
| Вывод | In conclusion, To sum up, All things considered |
| Уступка | Although, Even though, While it is true that |

### Для Lexical Resource (лексика)

Не повторять одно слово -- использовать синонимы:

| Базовое | Варианты |
|---|---|
| important | crucial, essential, significant, vital |
| problem | issue, challenge, concern, drawback |
| good | beneficial, advantageous, positive, favourable |
| bad | harmful, detrimental, negative, adverse |
| many | numerous, a significant number of, a great deal of |
| show | demonstrate, illustrate, indicate, reveal |
| think | believe, consider, argue, maintain |
| increase | rise, grow, surge, escalate |
| decrease | decline, drop, diminish, reduce |

### Для Grammatical Range (грамматика)

Экзаменатор хочет видеть разнообразие конструкций. Используйте:

| Конструкция | Пример |
|---|---|
| Passive | It is widely believed that... / Steps should be taken to... |
| Conditional | If governments invested more in education, the results would be significant. |
| Relative clause | Students who study abroad, which is becoming increasingly common, gain valuable experience. |
| Inversion | Not only does this save money, but it also helps the environment. |
| Cleft sentence | It is education that plays the most crucial role in development. |
| Gerund as subject | Learning a foreign language opens many doors. |
| Comparative | The more effort we put in, the better the results will be. |

## Типичные ошибки в IELTS Writing

| Ошибка | Штраф | Как избежать |
|---|---|---|
| Менее 150/250 слов | Сильный штраф по Task Achievement | Считать слова. Лучше 160/260 |
| Нет overview в Task 1 | Потолок 5.0 | Всегда второй абзац = overview |
| Нет чёткой позиции в Task 2 | Штраф по Task Achievement | Thesis в Introduction, подтверждение в Conclusion |
| Заученные фразы/эссе | Штраф по всем критериям | Использовать шаблоны структуры, не содержания |
| Одно время во всём эссе | Штраф по Grammar | Использовать разные времена где уместно |
| Повтор слов | Штраф по Lexical Resource | Использовать синонимы |
| Нет абзацев | Сильный штраф по Coherence | Минимум 4 абзаца |
| Перечисление без развития | Штраф по Task Achievement | Каждый аргумент: идея + пример + объяснение |
