# 🚀 Прокачка английского

Хаб про то, как системно качать английский дальше. Ядро словарного запаса у меня уже оцифровано (см. [[word-tracker|база 5000 слов]] → Anki). Здесь — направления роста и трекеры к ним.

> [!tip] Главный принцип
> Не «учить английский», а **жить в нём по чуть-чуть каждый день**: вход (слушать/читать) + выход (говорить/писать) + повторение (Anki). 🇺🇸 Везде держу фокус на американском произношении.

## Направления

### 1. 🎬 Обучение через контент → [[🎬 Что смотреть — дашборд]]
Фильмы, сериалы, мультсериалы и аниме (в английском дубляже) с упором на американский акцент. Курированный каталог по уровням A2→C1, со статусом «смотрю / в очереди / просмотрено» и обложками.
- **Как растить уровень:** мультсериалы (чистая дикция) → ситкомы (разговорная речь) → драмы и фильмы (быстрый темп, сленг).
- **Метод:** субтитры EN на A2–B1 → без субтитров на B2+; 3–5 новых слов за серию → в Anki; ключевые сцены пересматривай вслух за актёром (**shadowing**).

### 2. 🗂️ Активная лексика → [[word-tracker|база 5000 слов]]
2000+ слов с IPA (US), примерами и аудио, загружены в Anki с интервальными повторами. Новые слова из контента и чтения добавляй сюда же.

### 3. 🎧 Аудирование (подкасты)
Слушать в фоне/в дороге, американские:
- **American English Podcast** (Shana) — специально под изучающих, чёткая речь.
- **Culips**, **RealLife English** — разговорный американский, идиомы.
- **VOA Learning English** — новости упрощённым языком (A2–B1).
- Сложнее (B2+): любой нативный подкаст по интересам (NPR, Lex Fridman).

### 4. 📖 Чтение
- **VOA Learning English** / **graded readers** на старте.
- Дальше — статьи, субтитры из контента, книги по интересам.
- Новые слова → в базу/Anki.

### 5. 🗣️ Говорение и письмо (output)
- **Shadowing** по сценам из контента — лучшая прокачка произношения.
- Голосовой дневник: 2–3 минуты в день вслух о чём угодно.
- Письмо: веди заметки в Obsidian на английском; проверяй плагином **LanguageTool** (он у меня стоит).
- Практика с людьми: italki / Tandem.

### 6. 🔊 Произношение (американское)
- Ротичный R, T-flapping (water → «wader»), редукция гласных.
- Опора — IPA (US) в карточках Anki + повтор за актёрами из контента.
