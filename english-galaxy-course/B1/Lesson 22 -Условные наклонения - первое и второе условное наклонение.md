#### First Conditional
*If + Present Simple, Future Simple*
*When + Present Simple*

If it happens i will be happy
If I win the lottery i will buy something for myself
If it rains I won't go to the park
I'll tell you when I'm ready
He'll be very surprised when he knows it
If he earns enough money he will go to America

#### Second Conditional
*If + Past Simple + would/could + verb*
If only + Past Simple - если бы только + Past Simple

If he had more money he would buy a new house
If I were you, i would accept/take this offer
I wish I had a better memory
I wish I could spend more time on it
If only I knew it
If only I had more free time
If only I had this knowledge
If I were rich my life would be <font color="#c0504d">completely different</font>
